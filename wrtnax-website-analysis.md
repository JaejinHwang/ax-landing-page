## 1. 디자인 스타일 (Design System)

### 1.1 Typography

| 속성 | 값 |
|------|-----|
| **Primary Font** | `Pretendard Variable` (한국어 최적화 가변 폰트) |
| **Fallback Stack** | -apple-system, system-ui, Roboto, Helvetica Neue, Segoe UI, Apple SD Gothic Neo, Noto Sans KR, Malgun Gothic, sans-serif |
| **Icon Font** | `Material Symbols Rounded` (Material Design 아이콘) |
| **Code / Monospace** | `monospace` (기술 섹션 코드 표기) |

#### Font Size Scale

| 용도 | 크기 |
|------|------|
| Hero 헤딩 | 54px |
| 섹션 헤딩 (h2) | 40–48px |
| 서브 헤딩 (h3) | 28–32px |
| 소헤딩 (h4/h5) | 20–24px |
| 본문 | 16–18px |
| 캡션 / 라벨 | 12–14px |
| 최소 텍스트 | 10px |

#### Letter Spacing

| 값 | 용도 |
|------|------|
| `-0.96px` | 대형 헤딩 (타이트한 자간) |
| `-0.64px` | 중형 헤딩 |
| `-0.4px` | 소형 헤딩 |
| `-0.1px` | 본문 미세 조정 |
| `0.5px` / `0.8px` | 라벨, 대문자 캡션 (넓은 자간) |

### 1.3 Border Radius

| 값 | 용도 |
|------|------|
| `3px` | 소형 태그, 뱃지 |
| `4px` | 버튼, 인풋 기본 |
| `10px` | 아코디언 카드 |
| `16px` | 대형 카드 |
| `50%` / `100%` | 원형 아이콘, 스크롤 탑 버튼 |

### 1.4 Layout 특성

- **전체 다크 모드**: 페이지 전체가 어두운 배경 (`#0B0F11` ~ `#1E1E23`)
- **Single Page Application**: 앵커 링크 기반 섹션 네비게이션 (`#triangle`, `#education`, `#consulting` 등)
- **중앙 정렬 콘텐츠**: 최대 너비 제한 후 가운데 정렬
- **글래스모피즘**: `rgba(255,255,255, 0.03~0.1)` 반투명 카드 배경
- **총 페이지 높이**: ~13,550px (뷰포트 기준 약 14배)

## 2. 마이크로인터랙션 & 애니메이션

> 총 **45개의 인터랙션 CSS 규칙**과 **15개의 keyframe 애니메이션**이 감지되었습니다.

### 2.1 Hover 인터랙션

| 요소 | 동작 | 트랜지션 |
|------|------|----------|
| **네비게이션 링크 (`:hover`)** | `color` → `var(--cp1)` (브랜드 티얼 색상) | `color 0.25s` |
| **CTA 버튼 (`:hover`)** | `translateY(-2px)` 위로 살짝 올라감 + 티얼 글로우 그림자 | `background 0.35s, box-shadow 0.35s, transform 0.25s` |
| **문의하기 버튼 (`:hover`)** | 배경색 변경 + 글로우 box-shadow 추가 | `0.35s` |
| **아코디언 토글 (`:hover`)** | 아이콘 회전 (keyboard_arrow_down ↔ up) | `transform 0.3s` |
| **소셜 링크 (`:hover`)** | `transform: scale(1.1)` 확대 | 기본 트랜지션 |
| **일반 링크 (`:active`)** | 텍스트 데코레이션 제거 | - |

### 2.2 Transition 패턴

```css
/* 스크롤 등장 — 주요 패턴 */
opacity: 0.6s ease-out, transform 0.6s ease-out;       /* 기본 등장 */
opacity: 0.8s ease-out 0.2s, transform 0.8s ease-out 0.2s; /* 딜레이 등장 */
opacity: 1s ease-out, transform 1s ease-out;            /* 느린 등장 */
opacity: 0.5s ease-out, transform 0.5s ease-out;        /* 빠른 등장 */

/* 일반 인터랙션 */
color:              0.25s;
color:              0.3s;
opacity:            0.3s;
opacity:            0.5s;
opacity:            1s ease-in-out;
transform:          0.3s;
transform:          0.25s;

/* 복합 전환 */
background:         0.35s, box-shadow 0.35s, transform 0.25s;  /* CTA 버튼 */
height:             0.35s, opacity 0.25s, padding-bottom 0.35s; /* 아코디언 */
opacity:            0.5s, visibility 0.5s;                      /* 표시/숨김 */

/* 이징 함수 */
cubic-bezier(0.4, 0, 0.2, 1);  /* Material Design 표준 이징 */
ease-out;                       /* 주요 등장 이징 */
```

### 2.3 Keyframe Animations

| 애니메이션 이름 | 용도 |
|----------------|------|
| `fadeInUp` | 스크롤 진입 시 아래에서 위로 등장 (가장 핵심 애니메이션) |
| `cloudFloat` | Hero 배경의 빛줄기/구름 부유 효과 |
| `glowPulse1` | 글로우 펄스 효과 1단계 |
| `glowPulse2` | 글로우 펄스 효과 2단계 |
| `glowPulse3` | 글로우 펄스 효과 3단계 |
| `gradientShift` | 그라디언트 이동 애니메이션 |
| `waveFloat` | 웨이브 부유 효과 |
| `waveRotate` | 웨이브 회전 효과 |
| `slideInLeft` | 좌측에서 슬라이드 인 |
| `slideInRight` | 우측에서 슬라이드 인 |
| `spin` | 로딩 스피너 회전 |
| `blink` | 깜빡임 효과 |
| `lightbox-zoom-in` / `lightbox-zoom-out` | 이미지 라이트박스 확대/축소 |
| `overlay-menu__fade-in-animation` | 오버레이 메뉴 등장 |

### 2.4 스크롤 기반 인터랙션

- **Hero 배경**: 녹색 빛줄기(light beam) 그래픽이 `cloudFloat` 애니메이션으로 부유
- **글로우 펄스**: Hero 영역에서 `glowPulse1/2/3`로 빛이 은은하게 맥동
- **섹션 등장**: `fadeInUp` + `opacity/transform` 트랜지션으로 스크롤 시 콘텐츠 순차 등장
  - 기본: `0.6s ease-out`
  - 딜레이 적용: `0.8s ease-out 0.2s` (자식 요소에 stagger 효과)
  - 느린 등장: `1s ease-out` (배경 요소)
- **아코디언 (FAQ / 기술)**: `height 0.35s` + `opacity 0.25s` + `padding-bottom 0.35s`로 부드러운 확장/축소
- **Scroll-to-Top 버튼**: 우측 하단 고정, 원형 버튼 (`border-radius: 50%`), 티얼 보더
